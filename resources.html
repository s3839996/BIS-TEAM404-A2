Resources, Tools and Software

Microsoft Teams

Microsoft Teams will be the primary source of communications between group members, mentor, and the lecturer, and allows for all members to share work, ideas, and any questions they have. Virtual meetings can be held on here, which can also be recorded for future reference, by both members and teachers.

Canvas

Canvas will be the main source of any assignment updates, specifications, and other updates from the lecturer regarding the course. There is also a Canvas group that has been set up with all the members, but this will not be used throughout assignment development.


Trello

Trello is a tool used to replication the process of using a "Kanban Board", which is used as an agile project management tool to visualise work that needs to be completed. As such, all tasks that need to be completed are added to the groups Trello Board. When the status of a task changes, it can be reflected on the Trello Board. Trello is also used to indicate our weekly contributions at our weekly stand-up meetings.


Figma

Figma is another widely used tool in the industry, being used to create prototypes as to show off the end goal of the project. Figma will be used to develop all designs, concepts, as well as a fully functional prototype of our application. This will be managed entirely by Daniel.


LucidChart

LucidChart is yet another widely used tool in the industry, being used to create many types of diagrams, including flowcharts, use case diagrams, context diagrams, and plenty more that use UML notation. For our project, LucidChart will be used to collaborate with other members in creating any technical diagrams, mainly Use Case Diagrams and System Context Diagrams.


AWS

AWS will be used to host our website and all related files or media. Files can be downloaded from GitHub and uploaded to AWS in order to get an updated state of the entire website. This will be managed entirely by Michael.


Word Online

Word Online will be used to collaborate on most of the worded aspects of the assignment, which also allows members to ensure that wording is fluent and correct.


GitHub

GitHub is a widely used tool in the industry, particularly when it comes to programming and software development. It allows for members to upload any files, particularly HTML coding, whenever any significant changes have been. The updated versions can then be accessed by other members, whilst also allowing there to be the potential to go back to an earlier version in the event of any issues.
